## 🚀 Installation & Démarrage

### 📋 Prérequis
- **Roblox Studio** (version récente)
- **HttpService.HttpEnabled** = true
- **Allow HTTP Requests** activé
- **Team Create** recommandé pour collaboration

### ⚡ Installation Rapide
1. **Cloner le Repository**
   ```bash
   git clone https://github.com/fashion-revolution/roblox-game.git
   ```

2. **Importer dans Roblox Studio**
   - Ouvrir Roblox Studio
   - File → Open from File
   - Sélectionner le fichier `.rbxl` généré

3. **Configurer les Scripts**
   ```lua
   -- Dans ServerScriptService, créer un Script :
   local GameInitializer = require(game.ReplicatedStorage.FashionRevolution.GameInitializer)
   ```

4. **Lancer le Jeu**
   - Play → Start Server
   - Les systèmes s'initialisent automatiquement
   - Interface utilisateur moderne apparaît

### 🔧 Configuration Avancée
```lua
-- Dans GameInitializer.lua, modifier :
local GAME_CONFIG = {
    AI_GENERATION_ENABLED = true,      -- 🧠 IA Fashion Generator
    CINEMATIC_RUNWAY_ENABLED = true,   -- 🎪 Défilés cinématographiques
    CREATOR_ECONOMY_ENABLED = true,    -- 💰 Économie créateur
    BRAND_PARTNERSHIPS_ENABLED = true, -- 🤝 Partenariats marques
    ANALYTICS_ENABLED = true,          -- 📊 Analytics temps réel
    MAX_PLAYERS = 50,                  -- 👥 Limite serveur optimisée
    TARGET_FPS = 60                    -- ⚡ Performance cible
}
```
